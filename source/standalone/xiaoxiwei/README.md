# 小泰妍单文件运行时

开发者：个人非商业开发

`build.ps1` 生成 Windows 单文件 `小泰妍.exe`。程序把两套经过验证的 4K 角色资源直接嵌入程序集：

- `Taeyeon.Standalone.Frames.zip`：Weekend 粉蓝周末，176 张 `528×808` RGBA 关键帧及 313 组运动网格。
- `Taeyeon.Standalone.Invu.Frames.zip`：INVU 月影女神，176 张 `528×808` RGBA 关键帧及 313 组运动网格。

右键角色并打开“皮肤”，可以在 `Weekend 粉蓝周末（内置）` 与 `INVU 月影女神（内置）`之间切换。候选皮肤先在独立缓存中完整解码和预热，随后通过 60 阶段人物轮廓旋身、由头到脚揭示和原子缓存替换完成换装；选择保存在本地，下次启动自动恢复。

运行时特性：

- 无边框逐像素透明、置顶的 Windows 分层窗口。
- 固定裁切和基线的 4K RGBA 角色帧。
- 每个完整动画循环至少 24 个显示阶段。
- 光流形变只绘制一个不透明姿态，不使用整帧交叉淡化或双重残影。
- 拖动受惊与松手跺脚、双击挥手、滚轮缩放、16 方向鼠标视线跟随。
- 持续玩手机、侧躺入睡与头部锚定的 `z/Z` 粒子。
- 右键动作菜单、皮肤菜单、大小、暂停、置顶、托盘图标与单实例保护。
- 可选 API v1 第三方皮肤发现；缺失或无效的第三方皮肤不会影响两套内置皮肤。
- 当前用户级“开机自启”开关；不需要管理员权限。
- `--self-test` 自检和 `--qa-window` 限时 Windows 运行时 QA。

构建使用 .NET Framework 4.x 的 C# 5 编译器。运动网格只在构建期使用 `opencv-python-headless==4.12.0.88`；最终 EXE 不依赖 Python、OpenCV、DLL 或外部角色素材。
