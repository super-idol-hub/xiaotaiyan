# 小泰妍使用说明

## 基本操作

- 左键按住并拖动：移动角色。移动距离达到阈值后播放受惊反应；拖动期间保持受惊，松手后播放皱眉、抬脚、跺脚和抱臂生气。
- 单击睡眠中的角色：伸懒腰并起身；单击正在玩手机的角色：收起手机并起身。
- 双击：挥手。
- 鼠标靠近：眼神和头部小幅平滑跟随；移出近距离范围后立即回到普通呼吸眨眼待机。
- 鼠标滚轮：调整大小。
- 右键：打开动作、皮肤、公开动态提醒、置顶、暂停和退出菜单。
- `Ctrl+Q`：退出。

普通单击、双击、睡眠唤醒和手机退出有独立判定，不会被轻微抖动误触发。系统意外丢失鼠标捕获时也不会误触发跺脚。

## 随机待机与持续动作

安静一段时间后会随机播放特殊待机；最近两套动作不会连续重复，普通呼吸眨眼仍占主要时间。开始拖动会立即中止待机并恢复站立。

- 内置“Weekend 粉蓝周末”保留 r12 卖萌、r13 大笑、r14 哭和 r15“粉蓝周末星光亮相”。
- 内置“INVU 月影女神”把 r15 命名为“月影侧身行礼”。
- “坐下玩手机”进入后在低头、点击和滑动间持续循环，直到单击角色。
- “侧躺入睡”进入后保持睡姿，`z/Z` 从头部上方持续上飘，直到单击角色。

INVU 的 r15 按皮肤专属动作语义工作，不获得仅供 Weekend 兼容动作使用的特殊持续状态。

## 换肤过渡

右键角色并打开“皮肤”，菜单中固定提供：

- `Weekend 粉蓝周末（内置）`
- `INVU 月影女神（内置）`

两套皮肤均位于同一个 EXE 内，不需要复制外部资源。点击另一项即可切换，菜单勾选会跟随当前皮肤，选择会在重启后保留。

选择另一套皮肤后，程序只对人物 alpha 轮廓做水平切片和伪 3D 宽度/位移变化。新造型沿人物轮廓从头到脚逐段替换，随后定格 pose 并回到普通待机。完整过渡保持 60 个显示阶段、约 1.2 秒。

变身期间拖拽、双击和手动动作暂时锁定；暂停、缩放和关闭仍可安全执行。候选皮肤会先在独立缓存内完整解码、校验和预热，过渡完成时才原子替换当前缓存；无效皮肤不会破坏正在使用的造型。

## 公开动态提醒

- 沿用基线启用状态和低频检查行为，可在右键菜单随时关闭。
- 第一次检查只记录当前最新条目，不补发历史提醒。
- 之后检测到新条目时显示系统托盘提示；点击提示会打开经过可信 URL 校验的公开页面。
- “立即检查公开动态”用于手动触发一次后台检查，不会卡住动画。
- “开机自启”位于其下方，只写入当前 Windows 用户的启动项；取消勾选后会记住关闭状态。
- 公开页面可能延迟、改版或暂时不可用；断网时静默退避，不重复弹错。
