# 小泰妍 v1.1.1

本版本提供一个 Windows EXE、两套内置皮肤。

- 主程序：`小泰妍.exe`
- 内置皮肤一：Weekend 粉蓝周末
- 内置皮肤二：INVU 月影女神
- 切换方法：右键角色 → 皮肤 → 选择任一内置皮肤
- Weekend 专属动作：粉蓝周末星光亮相
- INVU 专属动作：月影侧身行礼
- 两套专属动作保持与普通动作一致的人物显示尺度
- 右键菜单“立即检查公开动态”下方提供“开机自启”
- 开发者：个人非商业开发

两套皮肤各自提供固定 24 行、176 张 `528×808` RGBA 关键帧及运动网格。完整循环至少呈现 24 个显示阶段；两套皮肤共用交互状态机，但人物造型与专属动作画面不同。

换肤候选资源会先完整验证、解码和预热，再通过 60 阶段人物轮廓旋身与由头到脚的揭示动画进行原子替换。当前皮肤选择会在重启后恢复。

最终 EXE 不依赖 Python、OpenCV、外置皮肤目录、DLL 或联网下载。公开照片、节目音视频、歌曲、Cookie、密钥和其他凭据均未打包。此项目仅供个人非商业使用，不是官方产品。
