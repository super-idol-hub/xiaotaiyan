# 小泰妍皮肤包模板

此模板预填官方外置皮肤 `taeyeon-invu`：

- 名称：INVU 月影女神
- 开发者：个人非商业开发
- r15 专属动作：月影侧身行礼

把完整 `frames.zip` 放入本目录。压缩包根目录必须直接使用 `r00/c00.png` 与 `motion/...` 路径；每张 PNG 固定为 `528×808` RGBA，完整提供 24 行、176 张关键帧。

如果复制模板制作第三方皮肤，请同步修改文件夹名、`skin.xml` 的 `id`、`name`、`developer` 和 `exclusiveAction`。`exclusiveAction` 非空时，r15 必须提供 8 张属于该造型、姿势连续的关键帧；留空时 r15 仅作兼容占位。

不得改变公共动作行、状态机、触发条件、时长、随机权重或安全上限。完整规则见 `docs/皮肤包接口.md`。
